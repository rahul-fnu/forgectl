<objective>
Fix the plumbing gap that makes all GitHub utility code in worker.ts dead code. The server.ts `onDispatch` callback discards octokit and repo (uses `_octokit, _repo`), and dispatcher.ts never constructs or passes GitHubDeps to executeWorker(). This single root cause blocks GHAP-03 (progress comments), GHAP-08 (check runs), and GHAP-09 (PR descriptions).

Purpose: Complete the wiring chain so that GitHub context flows from webhook handler through orchestrator to worker, enabling all existing GitHub utility code (comments, check runs, PR descriptions) to execute.
Output: Modified server.ts, orchestrator/index.ts, dispatcher.ts with working plumbing; unit tests.
</objective>

    - runDelegation checks depth >= 1 and returns empty outcome immediately (depth cap at 2 = lead + workers)
    - Each child gets a delegation row inserted with childRunId (crypto.randomUUID) BEFORE executeWorker is called
    - On child success: delegation row updated to 'completed' with result
    - On child failure (first attempt): rewriteFailedSubtask called, returns updated spec, retry dispatched
    - On child failure (retry also fails): child marked permanently failed, retryCount incremented
    - rewriteFailedSubtask invokes the agent with a rewrite prompt containing original issue + failure output
    - rewriteFailedSubtask returns null if rewrite agent invocation fails

    2. Implement `runDelegation`:

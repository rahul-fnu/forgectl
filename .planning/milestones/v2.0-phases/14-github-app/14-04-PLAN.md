         - "---"
         - "_Agent: {agent} | Workflow: {workflow}_"
         - "_Generated by forgectl_"
       - `updatePRDescription(octokit, owner, repo, prNumber, data: PRDescriptionData): Promise<void>` — calls octokit.rest.pulls.update({ body: buildPRDescription(data) })
